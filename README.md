# cumt-selection-in-advance

## 环境安装
cd 当前环境
安装虚拟python环境
```bash
python3 -m venv .venv
```

激活当前环境
```bash
.\.venv\Scripts\activate.bat
```
安装依赖库
```bash
pip install requests selenium webdriver-manager
```

## 运行代码
```bash
.\.venv\Scripts\activate.bat
python 提前选课.py
```

<font style="color:red">注意，登录后不要点×关闭，要在cmd中回车，窗口会自动关闭</font>

随后输入课程编码即可提前选课
